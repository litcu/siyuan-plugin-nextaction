层级规则

主标签页保留树形缩进和连接线；Dock 只显示父级/子级提示，不承担复杂树管理。

字段规则

标题最多两行，meta 最多两行；缺失字段不占位，过多字段用 chip 截断。

设计落点实现切分
主标签页

保留三栏结构，优化 NavRail、SearchFilterBar、TaskCard、TaskDetail，并为 MyDay/Review/Stats/Reminder 做特殊层级。

Dock

三 tab 轻量化，搜索更短，卡片更紧凑，MyDay 时间线保留拖拽和当前时间针。

TaskDetail

同一个组件要兼容右侧 pane 和 Dialog，通过 `dialogMode` 调整密度和页脚。

样式入口

基础 token 在 `tokens.scss`，全局结构在 `index.scss`，Dock 用局部 `:global` 压缩 TaskCard。
